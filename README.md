# Cell Aging Image Analysis

This project analyzes publicly available Cell Painting data to characterize cellular senescence phenotypes using CellProfiler morphology features.

## Project Goal

The goal is to explore whether image-derived morphology features can distinguish senescence-related compound treatments from control cells.

## Current Focus

- Understand the cleaned dataset structure
- Match compound IDs to drug names
- Explore MeasureObjectSizeShape features
- Compare DMSO/control cells with senescence-related treatments
- Evaluate filtering effects
- Build baseline classification models

## Project Structure

- `data/`: local data files, not tracked by Git
- `notebooks/`: exploratory analysis notebooks
- `src/`: reusable Python scripts
- `outputs/`: generated figures, tables, and model outputs
- `docs/`: project notes, meeting summaries, and questions

## Data

The raw and cleaned datasets are not tracked by Git due to size and privacy considerations.
